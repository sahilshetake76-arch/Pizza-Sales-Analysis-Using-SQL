# Pizza-Sales-Analysis-Using-SQL
This project analyzes pizza sales data using SQL to generate business insights related to customer demand, sales performance, product popularity, and revenue generation.

Business Questions Answered
• How many total orders were placed?
• What is the total revenue generated?
• Which pizza generates the highest revenue?
• What are the most popular pizza sizes?
• Which pizza categories perform best?
• What are the peak ordering hours?
• How does revenue change over time?
• Which pizzas contribute most to overall sales?

SQL Concepts Used
• Joins
• Aggregate Functions
• GROUP BY
• ORDER BY
• Subqueries
• Window Functions
• Ranking Functions
• Revenue Calculations

Key Outcomes
• Identified top-selling pizza types
• Measured category-wise performance
• Analyzed hourly ordering patterns
• Calculated revenue contribution by category
• Built cumulative revenue analysis
• Ranked top pizzas within each category

Tools
• MySQL
• SQL
• Data Analysis
• Business Intelligence
